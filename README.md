# Catálogo de Produtos

Catálogo de produtos com carrinho de compras desenvolvido com HTML, CSS e JavaScript, simulando o funcionamento de uma loja online simples.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- localStorage

---

## Funcionalidades
- Listagem dinâmica de produtos
- Adicionar produtos ao carrinho
- Atualização automática de quantidades
- Cálculo do valor total
- Limpar carrinho
- Persistência dos dados no navegador

---

## O que eu pratiquei neste projeto
- Manipulação avançada do DOM
- Arrays e objetos
- Lógica de carrinho de compras
- Organização de código
- Cálculo de valores
- Persistência de dados com localStorage
- Layout com CSS Grid
